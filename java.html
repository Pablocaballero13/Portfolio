üß© Controlador REST

Controlador que expone endpoints CRUD para gestionar usuarios desde la API.

Rutas: /api/users

@RestController
@RequestMapping("/api/users")
public class UserController {

  @Autowired
  private UserService service;

  @GetMapping
  public List<User> getAll() { return service.findAll(); }

  @PostMapping
  public User add(@RequestBody User user) { return service.save(user); }

  @PutMapping("/{id}")
  public User update(@PathVariable Long id, @RequestBody User user) {
    return service.update(id, user);
  }

  @DeleteMapping("/{id}")
  public void delete(@PathVariable Long id) {
    service.delete(id);
  }
}

üß† Capa Service

Contiene la l√≥gica de negocio y validaciones antes de guardar o modificar datos.

Tecnolog√≠as: Spring Service ¬∑ JPA

@Service
public class UserService {

  @Autowired
  private UserRepository repo;

  public List<User> findAll() { return repo.findAll(); }

  public User save(User user) { return repo.save(user); }

  public User update(Long id, User newUser) {
    return repo.findById(id).map(u -> {
      u.setName(newUser.getName());
      u.setEmail(newUser.getEmail());
      return repo.save(u);
    }).orElse(null);
  }

  public void delete(Long id) { repo.deleteById(id); }
}
