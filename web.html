🧩 Componente Navbar

Componente reutilizable que muestra una barra de navegación responsive con menú hamburguesa.

export function Navbar() {
  const nav = document.createElement('nav');
  nav.innerHTML = \`
    
Mi WebApp

    
    
    
      
  • Inicio
    • 
      
  • Sobre mí
    • 
      
  • Proyectos
    • 
    

  \`;

  const button = nav.querySelector('.menu-btn');
  const menu = nav.querySelector('.menu');
  button.addEventListener('click', () => {
    menu.classList.toggle('active');
  });

  return nav;
}

⚙️ Lógica Principal (SPA)

Gestión de navegación simple sin recargar la página, usando JavaScript nativo (sin frameworks).

// app.js
import { Navbar } from './components/navbar.js';

document.body.prepend(Navbar());

const routes = {
  home: "
Inicio
Bienvenido a mi SPA!
",
  about: "
Sobre mí
Alumno de 2º de DAM apasionado por el desarrollo web.
",
  projects: "
Proyectos
Aquí muestro mis trabajos más destacados.
",
};

window.addEventListener('hashchange', renderRoute);
function renderRoute() {
  const hash = window.location.hash.replace('#', '') || 'home';
  document.getElementById('root').innerHTML = routes[hash];
}
renderRoute();

🧪 Prueba Básica

Ejemplo simple de prueba unitaria con Jest para comprobar la generación del navbar.

// navbar.test.js
import { Navbar } from './navbar.js';

test('El componente Navbar crea un elemento nav', () => {
  const nav = Navbar();
  expect(nav.tagName).toBe('NAV');
});
